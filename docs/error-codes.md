# 错误码汇总

## 系统级错误码

| 错误码 | HTTP状态码 | 说明 |
|--------|-----------|------|
| 200 | 200 | 成功 |
| 400 | 400 | 参数校验失败 |
| 401 | 401 | 未认证或 token 无效 |
| 403 | 403 | 无权限访问 |

## 业务错误码

### 用户认证模块（10000-10099）

| 错误码 | 说明 |
|--------|------|
| 10001 | 邮箱已注册 |
| 10002 | 邮箱或密码不正确 |
| 10003 | 账户被禁止使用 |
| 10005 | 用户不存在 |

### 单词管理模块（20000-20099）

| 错误码 | 说明 |
|--------|------|
| 20001 | 单词重复创建 |
